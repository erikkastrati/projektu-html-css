Creative Hub’s success-oriented program is a combination of:

4 hours Personal Branding Workshop

For Developers LinkedIn is the #1 place to get noticed by recruiters and get hired. Every Academy in Creative Hub starts with Personal Branding Training, where the student gets his LinkedIn profile professionally optimized. They learn how the LinkedIn algorithm works and how to get noticed by recruiters on job postings. In this way, we set your profile for success prior the end of the Academy.

150 hours instructor-led sessions

The best part of being our student is that you get to choose whether you want to attend your lectures in our cozy camp; beautiful classes in Prishtina or from the comfort of your home. You get the chance to learn from the professional experience of our lecturers. They will transfer you practical knowledge and you will get homework assignments on which they will give you feedback.

Practical Projects

We host Masterclasses, Hackathons, or Visiting Lectures with professionals from big and well- known companies in Kosovo every month. This is a great opportunity for you to network with accomplished people from the industry and to collaborate with them to build projects for your portfolio. 97% of our students rated these events 10/10 and said this is the best opportunity that was given to their careers as programmers.

8 hours Hire-Me Program

Usually, other Academies or trainings end when the student gets the certificate, at Creative Hub the Academy ends when the student gets a job or a paid internship.  As part of the program, after finishing with the curriculum you will continue with 8 hours of training with a coach that will help you to build your CV/Git Portfolio/Elevator Pitch and will follow your progress until you find a job.

For this to successfully happen, the student should also put effort during classes and complete his assignments and projects

Curriculum

The Academic Program lasts 12 months.

Student will learn

  • Introduction to web development and HTML
  • HTML syntax and structure
  • HTML tags and elements
  • Working with links, images, and forms

Student will learn

  • Introduction to CSS and its role in web development CSS
  • CSS syntax and selectors
  • Basic CSS properties for styling HTML elements
  • Creating a basic stylesheet Working with layout and positioning

Student will learn

  • How to use bootstrap
  • Layouts and Grids
  • Typo and Colors
  • Working with Forms and Inputs
  • Using Components

Student will learn

  • Create a git account
  • Installing GitHub
  • Creating a new Git repository
  • Working with Branches
  • Git pull, push, merge etc.

Student will learn

  • Setting up a PHP development environment
  • Variables, data types, operators, and expressions
  • Control structures (if/else, loops)
  • Functions and arrays
  • PHP forms and user input

Student will learn

  • Object-oriented programming concepts (classes, objects, inheritance, polymorphism)
  • Implementing OOP in PHP
  • PHP exceptions and error handling
  • PHP debugging techniques

Student will learn

  • Introduction to MySQL
  • ER Diagram
  • Indexing and data types
  • Database Relations
  • CRUD operations in PHP

Student will learn

  • Introduction to MySQL
  • Introduction to Laravel framework and its features
  • Laravel installation and setup
  • Laravel application structure and directory layout
  • Routing in Laravel

Student will learn

  • Laravel database migrations
  • Database seeding in Laravel
  • Working with Eloquent ORM in Laravel
  • Querying data in Laravel

Student will learn

  • Following MVC pattern by using blade templates
  • Creating Models, Views, Controllers
  • Using Forms and Validations
  • Learning how to research on the internet for different problems
  • Using Google, Stackoverflow, ChatGPT

Student will learn

  • User authentication in Laravel (registration, login, logout)
  • Laravel authorization (using gates and policies)
  • User roles and permissions in Laravel
  • Social authentication in Laravel

Student will learn

  • Short introduction to jQuery
  • Selectors and Forms
  • Using Ajax Requests
  • Data parsing

Student will learn

  • Building RESTful APIs in Laravel
  • API authentication and security
  • API versioning and documentation
  • Consuming APIs with Laravel

Student will learn

  • What is Composer and why we need
  • Use composer package in our project
  • Install new composer package and use it

Student will learn

  • Developing a Laravel project from scratch
  • Applying concepts and techniques learned throughout the course
  • Planning, designing, and implementing a functional web application
  • Deploying and maintaining a Laravel project
